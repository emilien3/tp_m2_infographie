# UTILISATION
# =====================================

0. git clone https://github.com/libigl/libigl.git
1. Créez un dossier build: mkdir build && cd build
2. Configurez: cmake ..
3. Compilez: make -j4 (Linux/Mac) ou cmake --build . (multiplateforme)
4. Exécutez: ./mesh_tp
